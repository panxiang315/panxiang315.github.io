console学习笔记

发表于 2018-07-20 | 分类于 programme > HTML
字数统计 414字 | 阅读时长 2分

  很多厉害的导演喜欢在电影中使用长镜头，一镜到底，是非常考验功力的。代码中也是一样，玩一镜到底很容易玩脱，这时候就要一些中间点来辅助我们，可以引入console控制台。

强大的console

简单显示

console.clear() 清空调试窗口

console.log(“x”) 简单显示

console.debug(“x”) 显示为蓝色字体

console.info(“x”) 显示为带感叹号圆形

console.warn(“x”) 显示为带感叹号三角形

console.error(“x”) 显示为带叉圆形

console.assert(bool,”x”) 当bool为false时才输出

以上显示信息的方法都可以使用printf风格的占位符。不过，占位符的种类比较少，只支持字符（%s）、整数（%d或%i）、浮点数（%f）和对象（%o）四种。
如 console.log(“圆周率是%f”,3.1415926);

高级显示

console.dir(obj) 将对象以树状显示

console.dirxml() 与getElementById(“idname”)组合使用，如下：

console.group(“groupid”)与console.groupEnd()组合使用，形成一组显示，中间可以放一些console.info/log等语句
console.groupCollapsed(“groupid”)与console.groupEnd()组合使用类似，只是它会直接是收起来的

console.table(objs) 用表格的格式显示多个obj属性

统计计算

console.count(“x”) 统计该语句被执行的次数，参数是“x”

console.profile(“a”)与console.profileEnd(“a”)组合使用，可以看到期间的CPU使用情况

console.time(“a”)与console.timeEnd(“a”)组合使用，可以得出执行这中间代码的时间

console.timeline(“a”)与console.timelineEnd(“a”)组合使用，可以记录时间轴

调试

console.trace 堆栈跟踪调试
